# bookstore-app

A bookstore application where store owner can create, search, update and delete books.

### Features
Store owner can:
- Create Books
- Fetch Books
- Update Books
- Delete Books
- Search for books

### Tech Stack

**Server:** Node, Express, Mongoose, MongoDB
